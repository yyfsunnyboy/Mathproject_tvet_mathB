# 自適應複習模式 - 使用指南

## 📋 概述

自適應複習模式是一個基於 **AKT（注意力知識追蹤）** 和 **RL（強化學習）** 的智能複習系統。它根據學生的學習歷史，動態推薦最適合的複習題目，幫助學生高效地鞏固知識。

### 核心特點
- ✅ **智能推薦**：基於 RL 模型推薦最優題目序列
- ✅ **知識追蹤**：使用 AKT 模型實時追蹤學生的掌握度
- ✅ **簡潔界面**：只保留必要的功能元素，避免過度複雜
- ✅ **多模式支持**：手寫板、AI 助手、即時反饋

## 🎯 用戶界面

### 布局

```
┌─────────────────────────────────────────────────────┐
│  【當前 APR】  【熟練分數】  【重新開始】【返回】  │
├──────────────────────────────────────────────────────┤
│              │                  │                    │
│  手寫板      │  題目&答案        │   AI 助手          │
│              │                  │                    │
│  ✏️ 工具     │  📝 題目          │  💬 對話歷史      │
│  🧹  橡皮擦   │  ✍️ 答案輸入      │  📝 輸入框        │
│  🗑️ 清除     │  💡 提示          │                    │
│              │  ⏭️ 跳過          │                    │
│              │  ➡️ 下一題        │                    │
└──────────────────────────────────────────────────────┘
```

### 主要功能區

#### 左側：手寫板 ✏️
- **筆工具**：在板上手寫題目的計算過程
- **橡皮擦**：擦除錯誤的手寫內容
- **清除**：一鍵清除整個手寫板

#### 中間：題目 & 答案 📝
- **RAG 模式指示**：顯示系統目前使用的模式
- **題目顯示**：顯示當前需要回答的題目
- **技能標籤**：顯示題目考查的技能
- **難度指示**：簡單/中等/困難
- **答案輸入框**：輸入你的答案
- **提交按鈕**：提交答案以獲取反饋
- **快速按鈕**：
  - 💡 **提示**：獲得 AI 提示
  - ⏭️ **跳過**：跳過當前題目
  - ➡️ **下一題**：加載下一道推薦題目

#### 右側：AI 助手 🤖
- **對話歷史**：查看與 AI 的所有對話
- **實時消息**：AI 的即時反饋和建議
- **快速提問**：輸入你的問題並發送

#### 頂部：統計信息 📊
- **當前 APR**：你當前的平均掌握度（百分比）
- **熟練分數**：轉換後的 1-100 分熟練度評分

## 📚 工作流程

### 典型的複習會話

```
1️⃣  開啟應用
    ↓
2️⃣  系統載入推薦題目
    ├─ 基於你的學習歷史分析當前掌握度 (APR)
    └─ 使用 RL 模型推薦最優題目
    ↓
3️⃣  查看題目
    ├─ 理解題目要求
    └─ 在手寫板上計算（可選）
    ↓
4️⃣  輸入答案
    ├─ 在「答案輸入框」中輸入答案
    └─ 按「提交」按鈕
    ↓
5️⃣  收到反饋
    ├─ 系統更新你的掌握度
    ├─ 顯示正確/錯誤反饋
    └─ AI 助手提供解釋
    ↓
6️⃣  下一題
    ├─ 系統推薦新題目
    └─ 返回第 3 步，重複練習
    ↓
7️⃣  完成會話
    └─ 當達到目標掌握度 (80%) 或滿足會話要求時結束
```

## 🔧 API 端點

### 1. 開始複習會話
**POST** `/api/adaptive-review/start`

**請求示例：**
```json
{
  "student_id": "STU_001",
  "history": {
    "item_history": [1, 2, 3],
    "skill_history": [0, 1, 2],
    "resp_history": [1, 1, 0]
  },
  "n_recommendations": 1,
  "use_rl": true
}
```

**響應示例：**
```json
{
  "status": "success",
  "data": {
    "current_apr": 0.68,
    "recommendations": [
      {
        "item_id": 42,
        "skill_id": 3,
        "skill_name": "polynomial-operations",
        "predicted_difficulty": 0.45
      }
    ]
  }
}
```

### 2. 提交答題反饋
**POST** `/api/adaptive-review/feedback`

**請求示例：**
```json
{
  "student_id": "STU_001",
  "history": {...},
  "item_id": 42,
  "correct": 1,
  "time_spent": 45
}
```

**響應示例：**
```json
{
  "status": "success",
  "data": {
    "apr_before": 0.68,
    "apr_after": 0.71,
    "apr_change": 0.03,
    "feedback": "很好！掌握度提升了...",
    "next_recommendations": [...]
  }
}
```

### 3. 分析學生弱項
**GET** `/api/adaptive-review/analyze/<student_id>`

查詢參數：`history_json` (URL-encoded JSON)

**響應示例：**
```json
{
  "status": "success",
  "data": {
    "overall_apr": 0.62,
    "weakest_skills": [
      {
        "skill_id": 5,
        "skill_name": "radical-operations",
        "mastery": 0.35
      }
    ]
  }
}
```

### 4. 獲取題目詳情
**GET** `/api/adaptive-review/question/<item_id>`

**響應示例：**
```json
{
  "status": "success",
  "data": {
    "item_id": 42,
    "skill_id": 3,
    "question_text": "化簡多項式...",
    "predicted_difficulty": 0.45,
    "hint": "可以先分組..."
  }
}
```

### 5. 健康檢查
**GET** `/api/adaptive-review/health`

用於驗證服務是否正常運行。

```python
REVIEW_CONFIG = {

    'max_session_length': 15,        # 最多出題數

    'review_diversity_weight': 0.3,  # 多樣性權重
}

BETA_THRESHOLD = 0.65  # 目標掌握度 (65%)
```

## 📊 APR (Average Posterior Responsibility)

**APR** 是系統評估你掌握度的關鍵指標。

### 計算方式
- 基於 AKT（注意力知識追蹤）模型
- 考慮所有答題記錄的時間衰減
- 返回 0 到 1 之間的值

### 轉換公式
```
熟練分數 = APR × 85
```

### 例子
- APR = 0.60 → 熟練分數 = 51
- APR = 0.75 → 熟練分數 = 64
- APR = 0.80 → 熟練分數 = 68（達到目標）

## 🐛 常見問題

### Q: 如何查看我的學習歷史？
**A:** 系統會自動記錄你的所有答題。在複習頁面頂部可以看到當前的 APR。

### Q: 為什麼題目難度會變化？
**A:** 系統根據你的表現動態調整難度：
- 連續答對 → 增加難度
- 連續答錯 → 降低難度

### Q: 可以跳過不想做的題目嗎？
**A:** 可以！按「⏭️ 跳過」按鈕即可加載下一題。跳過的題目不會計入記錄。

### Q: AI 助手能做什麼？
**A:** 
- 提供題目解釋
- 給出學習建議
- 回答你的問題
- 提供學習策略建議

### Q: 達到目標後會怎樣？
**A:** 當 APR 達到目標掌握度 (65%) 時：
1. 系統會提示你已達到目標
2. 可以選擇繼續練習進階題目
3. 或結束當前會話

## 📈 進度追蹤

### 在複習會話中

頂部顯示的統計信息會實時更新：

| 指標 | 含義 | 更新頻率 |
|------|------|--------|
| 當前 APR | 你的平均掌握度 | 每提交一道題目 |
| 熟練分數 | 轉換後的 1-100 分 | 每提交一道題目 |

### 長期進度

可以通過以下方式查看長期進度：
1. 訪問個人儀表板
2. 查看學習統計報告
3. 對比不同時期的 APR 變化

## 🚨 故障排除

### 應用無法啟動
```bash
# 1. 檢查 Python 版本
python --version  # 需要 Python 3.8+

# 2. 安裝依賴
pip install -r requirements.txt
```

### API 返回錯誤
- 檢查歷史數據格式是否正確
- 確保 `item_history`、`skill_history`、`resp_history` 長度相同
- 檢查網絡連接

### 題目無法加載
- 刷新頁面
- 清除瀏覽器緩存
- 檢查應用日誌
