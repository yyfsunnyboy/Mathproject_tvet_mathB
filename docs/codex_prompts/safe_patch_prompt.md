# Safe Patch Prompt

Follow `AGENTS.md` and apply the smallest safe patch only.

## Non-Negotiable Constraints
- No architecture changes.
- No new skill creation.
- No unrelated module impact.
- Minimal file/line footprint.

## Allowed Behavior
- Targeted edits in approved files only.
- Focused verification only for impacted scope.

## Required Confirmation
- Confirm AGENTS.md compliance.
- Show exact changed files.
- Confirm runtime impact is none/minimal.
