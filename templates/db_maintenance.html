資料庫全域操作 (Database Operations)
請謹慎使用以下功能，特別是「匯入」與「清空」功能。操作前建議先進行備份（匯出）。

備份資料庫

下載當前的資料庫所有表格為 Excel 檔案 (.xlsx)。可用於備份或遷移數據。

core 只匯出教材核心表；full 才匯出全部資料表。

資料匯入 (Excel/CSV)

Excel (.xlsx) 模式： 每個 Sheet 名稱需對應資料庫 Table 名稱 (例如: users, skills_info)。
若 Sheet 名稱在資料庫中找不到，將會被自動略過。
core 只匯入核心表；full 允許全表匯入（需確認字串）。

清空教材資料（core）

只清空教材核心表，保留 prompt、system settings、API Key、使用者與紀錄資料。

清空全部資料（full）

完整清空會刪除 prompt_templates、system_settings、API Key、使用者與紀錄資料。
需輸入 YES_DELETE_ALL 才可執行。
