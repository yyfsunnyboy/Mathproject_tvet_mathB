## 🎮 基本操作

### 解題流程

1. **查看題目** 📖
   - 題目會自動在中間面板顯示
   - 注意技能標籤和難度指示

2. **計算過程** ✏️
   - 在左側手寫板上寫下計算過程
   - 使用 ✏️ 筆工具或 🧹 橡皮擦

3. **輸入答案** ✍️
   - 在「答案輸入框」中輸入最終答案
   - 點擊「提交」按鈕

4. **獲得反饋** 💬
   - AI 助手會提供即時反饋
   - 你的掌握度 (APR) 會自動更新

5. **下一題** ➡️
   - 自動或手動加載下一道推薦題目
   - 重複上述流程

### 快速操作

| 功能 | 按鈕 | 說明 |
|------|------|------|
| 獲取提示 | 💡 | AI 助手會給出解題思路 |
| 跳過題目 | ⏭️ | 不想做的題目可以跳過 |
| 下一題 | ➡️ | 手動加載下一道題 |
| 清除手寫 | 🗑️ | 擦除手寫板上的內容 |
| 重新開始 | 🔄 | 重置整個複習會話 |

## 📊 理解統計信息

### 當前 APR（平均掌握度）
- **含義**: 你對當前技能的掌握程度
- **範圍**: 0% 到 100%
- **目標**: 達到 65% 或以上

### 熟練分數
- **計算方式**: APR × 85
- **範圍**: 0 到 85
- **例子**: 
  - APR 60% = 熟練分數 51
  - APR 75% = 熟練分數 64
  - APR 80% = 熟練分數 68

## 💡 使用技巧

### 最大化學習效果

1. **集中注意力**
   - 一次只做一道題
   - 先自己思考，再看提示

2. **充分利用手寫板**
   - 記錄解題步驟
   - 有助於複習時回顧

3. **和 AI 助手互動**
   - 提出問題
   - 尋求解釋

4. **定期複習**
   - 每天練習 5-10 道題
   - 利用系統的推薦功能

### 高效率的複習

- 花 60% 時間做推薦題目
- 花 20% 時間做有挑戰的題目
- 花 20% 時間複習之前的內容

## ⚙️ 常見設置

### 調整手寫板

- **筆工具** ✏️：用於手寫
- **橡皮擦** 🧹：修正錯誤
- **清除** 🗑️：一鍵清空

### 調整難度

系統會根據你的答題情況自動調整：
- 答對 ➜ 下道題目難度增加
- 答錯 ➜ 下道題目難度降低

### 調整速度

- 沒有時間限制
- 可以暫停重新整理思路
- 隨時可以查看提示

pip install -r requirements.txt

- 確認已輸入答案
- 檢查應用是否還在運行

## ✨ 最佳實踐

### 每日複習計畫

**推薦時長**: 20-30 分鐘

```
5 分鐘  - 複習上次的弱項 (1-2 題)
15 分鐘 - 做推薦題目 (5-7 題)
5 分鐘  - 查看統計和進度
5 分鐘  - 與 AI 助手討論難點
```

### 每週複習計畫

```
週一: 基礎技能複習 (APR < 60%)
週二: 進階技能練習 (APR 60-75%)
週三: 混合練習 (新舊知識結合)
週四: 難題挑戰 (最困難的題目)
週五: 綜合複習
週六: 弱項加強
週日: 放鬆複習或休息
```
